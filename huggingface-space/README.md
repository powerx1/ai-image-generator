## How to Use

1. Enter a descriptive prompt
2. Adjust settings (steps, CFG scale, size)
3. Click Generate!

## Tips

- Use descriptive prompts: "A beautiful sunset over mountains, digital art, highly detailed"
- Add style keywords: "oil painting", "watercolor", "photorealistic", "anime style"
- Use negative prompts to avoid unwanted features
