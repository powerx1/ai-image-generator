Activity Diagram

AI-Based Image Generator and Processor Use Case Flow

Legend

Oval: Start/End points
Rectangle (Rounded): Activities/Actions
Diamond: Decision points
Arrow: Flow direction

Activity Flow Description

  • Authentication Phase: User registers or logs in to access the system
  • Activity Selection: User chooses between Text-to-Image, Image-to-Image, or Image-to-Text
  • Input Phase: User provides necessary input (text prompt or image upload)
  • Processing Phase: System forwards request to AI model via API
  • Result Display: Generated output is displayed to user
  • Preview & Save: User can preview and optionally save/download the result
  • Iteration: User can generate more images or end session
